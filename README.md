# 🚀 SpaceX Falcon 9 Landing Prediction

This repository contains my end-to-end data science project developed as part of the IBM Data Science Professional Certificate. The project aims to predict the successful landing of Falcon 9 first-stage boosters using historical launch data and machine learning techniques.

## 📂 Repository Structure

```text
MyProject
│
├── Part 1 - Data Collection & Wrangling
├── Part 2 - SQL & Data Visualization
├── Part 3 - Interactive Visual Analytics
├── Part 4 - Machine Learning
└── Part 5 - Final Report
```

## 📌 Project Workflow

1. Collect launch data from the SpaceX API
2. Scrape launch records from Wikipedia
3. Clean and preprocess the dataset
4. Explore data using SQL and visualizations
5. Build interactive dashboards and maps
6. Train and evaluate machine learning models
7. Present findings in a final report

## 📁 Repository Contents

### 📦 Part 1 — Data Collection & Wrangling

- Collect data from the SpaceX REST API
- Web scrape launch records from Wikipedia
- Clean and merge datasets
- Prepare data for analysis

### 📊 Part 2 — SQL & Data Visualization

- Query data using SQLite
- Perform exploratory data analysis
- Visualize relationships and trends
- Generate insights through statistical graphics

### 🌍 Part 3 — Interactive Visual Analytics

- Create Folium maps
- Develop an interactive Dash dashboard
- Analyze launch success by payload, orbit, and launch site

### 🤖 Part 4 — Machine Learning

- Feature engineering
- Data preprocessing
- Model training and evaluation
- Hyperparameter tuning with GridSearchCV
- Compare multiple classification models

### 📄 Part 5 — Final Report

- Project presentation
- Dashboard summary
- Model evaluation
- Conclusions and recommendations
